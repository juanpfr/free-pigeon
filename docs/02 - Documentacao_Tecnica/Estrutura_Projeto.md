### 🧰 Tecnologias Utilizadas
- **Backend:** Django, Django REST Framework, Celery, Redis  
- **Banco de Dados:** PostgreSQL  
- **Frontend:** Vue.js, Tailwind CSS  
- **Integrações:** Stripe, AWS S3 (django-storages, boto3)
